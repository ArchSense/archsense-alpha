# Migrate to Vite

## Context and Problem Statement

Create-react-app is not supported by the community anymore. Its configuration is quite cumbersome.


## Decision Drivers

* Modern actively-supported tool
* Native react support
* Easy of config

## Considered Options

* Vite
* Remix

## Decision Outcome

Vite.
